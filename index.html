    WAFFLE - Windows Authentication Framework

    WAFFLE is a native Windows Authentication Framework consisting of two C# and Java libraries that perform functions relat ed to Windows authentication, supporting Negotiate, NTLM and Kerberos. Waffle also includes libraries that enable drop-i n Windows Single Sign On for popular Java web servers, when running on Windows. While Waffle makes it ridiculously easy to do Windows Authentication in Java, on Windows, Waffle does not work on *nix.
